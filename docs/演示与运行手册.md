# 演示与运行手册

## 启动与关闭

1. 首次运行 04_配置AI密钥.cmd。
2. 运行 01_启动作品.cmd。
3. 浏览器会打开产品工作台和后端健康状态。
4. 演示结束运行 02_关闭作品.cmd。

## 演示路径

1. 先运行发现阶段，查看公开证据和候选工作流。
2. 对比评分维度，选择一个候选。
3. 运行深度拆解，查看痛点和 Agent 介入点。
4. 打开原流程图、Agent 流程图、风险审查和最终报告。
5. 查看 trace、metrics 与 llm_usage，证明真实调用。

## 故障排查

- 7860 被占用：运行 03_检查状态.cmd，确认占用 PID。
- 没有真实模型：重新运行 04_配置AI密钥.cmd。
- 搜索不可用：系统会回退证据池，结果中应显示 source_mode。
- 启动失败：检查 .runtime 中的 stdout 和 stderr 日志。
